# Uproszczona wersja projektu GA

Ta wersja upraszcza projekt do 4 plików:
- `ga_engine.py` – cały silnik algorytmu
- `gui_app.py` – proste GUI w Tkinterze
- `results_io.py` – zapis CSV i JSON
- `main.py` – start aplikacji

## Co dalej obsługuje
- funkcję Rosenbrocka,
- binarną reprezentację,
- selekcję: best / roulette / tournament,
- krzyżowanie: single-point / two-point / uniform / granular,
- mutację: edge / single-point / two-point,
- inwersję,
- elitę,
- GUI,
- wykres,
- zapis wyników.

## Uruchomienie
```bash
pip install -r requirements.txt
python main.py
```
